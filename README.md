
[JSP]게시판(글,댓글,페이징,파일업로드) 구현 프로젝트
===
--------------------------------------

* #### 프로젝트 소개: 백엔드는 Spring Boot를 이용하고 프론트엔드는 JSP,Ajax,BootStrap 활용하여 게시판의 기능(게시글,댓글,파일업로드)를 구현한다.
----------------------------------

### 사용 기술

* #### 백앤드: 
   * JPA(Spring Data JPA)
   * Java 19
   * SpringBoot 3.1.2
   * Swagger 2.1

* #### 프론트앤드:
  * JSP
  * JavaScript
  * Bootstrap

* #### DataBase
  * MySQL 8.0

* #### Build Tool: 
  * Gradle 7.6
